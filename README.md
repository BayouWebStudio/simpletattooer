# Simple Tattooer Landing Page

A modern SaaS landing page for Simple Tattooer - stress-free tattoo admin software.

## Overview

**Simple Tattooer** is a SaaS product designed to automate administrative tasks for tattoo artists, including:
- Automated client emails
- Smart booking management  
- Client management system
- Deposit tracking

This landing page is built as a demo for BWS (Bayou Web Studio) showcasing modern SaaS design principles.
